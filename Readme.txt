Description\
==============\
MyMiniDesktop is a fully functional desktop application designed to create\
and maintain a self-contained flat file database of MSDS (or other) compound\
information. It also contains an in-application calculator, notepad, and unit\
converter.\
\
==============\
Requirement:\
==============\
MyMiniDesktop requires at least Java Runtime Environment (JRE) version 6.0.0\
\
==============\
run the program\
==============\
In order for the program to run you will need the files compound.jar, guiMyMSDS.jar, jxl.jar and the folder datafile in the same directory.\
open the file "run.jar"  to open the program}
